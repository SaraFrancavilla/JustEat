## Istruzioni
#### Come far connettere a giocare il blob A
1. Importa tutte le modifiche sulla branch "llm"
2. Ricorda di fare l'accesso alla VPN GlobalProtect
3. Vai nella directory agentA e nel terminale digita:
    ```bash
   npm start
   ```
4. Crea il tuo blob, connettilo, copia il token inseriscilo nel terminale
5. Guarda il blob
