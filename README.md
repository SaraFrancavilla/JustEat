## Istruzioni
#### Come far connettere a giocare il blob
1. Importa tutte le modifiche sulla branch "llm"
2. Ricorda di fare l'accesso alla VPN GlobalProtect
2. Nel terminale digita
    ```bash
   npm start
   ```
3. Crea il tuo blob, connettilo, copia il token inseriscilo nel terminale
4. Guarda il blob

#### Come dare challenge al blob per testarlo
1. Creare un altro blob con il nome ChallengeGiver
2. Scrivere delle 'challenge' nella chat con il nuovo blob
