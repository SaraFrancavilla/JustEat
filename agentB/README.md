## Istruzioni

#### Come far connettere a giocare il blob B

1. Importa tutte le modifiche sulla branch "llm"
2. Ricorda di fare l'accesso alla VPN GlobalProtect
3. Vai nella directory agentB e nel terminale digita:

    ```bash
   npm start
   ```

4. Crea il tuo blob, connettilo, copia il token inseriscilo nel terminale
5. Guarda il blob

#### Come dare challenge al blob per testarlo
1. Creare un altro blob (di tipo A o B) con il nome ChallengeGiver o Professor
2. Scrivere delle 'challenge' nella chat con il nuovo blob
